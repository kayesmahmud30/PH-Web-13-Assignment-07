# 👥 KeenKeeper — Keep Your Friendships Alive

KeenKeeper is a simple friendship management web app that helps users keep track of their friends and stay connected with them. The application allows users to monitor communication gaps, perform quick check-ins, and view interaction history through a timeline and analytics.

---

# 📌 Project Overview

Maintaining friendships can be difficult when life gets busy. **KeenKeeper** helps solve this problem by tracking when you last contacted your friends and reminding you to stay in touch.

Users can:

- View their friends and relationship status

- Log interactions like calls, texts, and video chats

- Track interaction history through a timeline

- Analyze communication patterns using charts

---

# 🛠️ Technologies Used

This project was built using modern frontend technologies:

- **React.js** – UI development

- **React Router DOM** – Page navigation and routing

- **Tailwind CSS** – Styling and responsive design

- **DaisyUI** – UI component library

- **React Icons** – Icons for UI elements

- **React Toastify** – Toast notifications

- **Recharts** – Data visualization (Pie Chart)

- **Vercel** – Deployment and hosting

---

# ✨ Key Features

### ⚡ Quick Check-In System

Users can quickly log interactions:

- 📞 Call

- 💬 Text

- 🎥 Video

Each action automatically creates a **timeline entry** and shows a **toast notification**.

### 📜 Timeline History

- View all previous interactions in one place

- Shows **date, interaction type, and title**

- Filter interactions by **Call / Text / Video**

### 📊 Friendship Analytics

- Pie chart visualization of interaction types

- Displays the distribution of **Calls, Texts, and Videos**
